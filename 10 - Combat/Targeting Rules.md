# Targeting Rules

## MVP Target Types
- Enemy single target
- Ally single target (for heals/buffs)
- Self (guard, self-buff)

## Valid target filters
- Living only
- Optional: Taunt redirects (future)
- Optional: “frontline only” (future)

## UI note
Target selection should be separate from action selection:
- Choose Action → Choose Target → Confirm
