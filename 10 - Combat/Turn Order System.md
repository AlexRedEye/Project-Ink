# Turn Order System

## MVP Rule
- Order actors by **SPD** descending.
- Skip defeated units.
- Loop continuously.

## Future Extensions (optional)
- Tie-breakers (stable order / initiative roll)
- Haste/Slow effects (see [[10 - Combat/Status Effects System]])
- Timeline system (delay, speed scaling)
